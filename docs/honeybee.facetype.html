class honeybee.facetype.AirWall[source]¶

Bases: honeybee.facetype._FaceType

Type for virtual partitions between Rooms.

Note that the use of the word ‘Wall’ in AirWall does not limit the application of this type to vertical faces. It can be applied to any face between two Rooms.

honeybee.facetype.get_type_from_normal(normal_vector, roof_angle=30, floor_angle=150)[source]¶

Return face type based on the angle between Z axis and normal vector.

Angles between 0 and roof_angle will be set to roof_ceiling. Angles between roof_angle and floor_angle will be set to wall. Angles larger than floor angle will be set to floor.

Parameters:
  • normal_vector – Normal vector as a ladybug_geometry Vector3D.
  • roof_angle – Cutting angle for roof from Z axis in degrees (default: 30).
  • floor_angle – Cutting angle for floor from Z axis in degrees (default: 150).
Returns:

Face type instance.
