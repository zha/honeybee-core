honeybee.logutil module¶

honeybee.logutil.get_logger(name, filename='honeybee.log', file_log_level='DEBUG', console_log_level='WARNING')[source]¶

Get a logger to be used for each module.

Parameters:
  • name – Logger name. The good practice is to set it to __init__ from inside each modules.
  • filename – Logger filename.Setting filename to None will remove the file handler (Default: honeybee.log).
  • file_log_level – Log level for file handler as a string (Default: DEBUG).
  • console_log_level – Log level for stream handler as a string (Default: WARNING).
